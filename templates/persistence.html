What This Page Shows
Persistence analysis is part of Topological Data Analysis. It helps show whether the dataset has strong clusters, connected structures, or possible holes/patterns in the embedding space.
Persistence Diagram

Shows when topological features appear and disappear. Points that are far from the diagonal usually represent stronger or more persistent structures.

Persistence Barcode

Shows the lifetime of each topological feature as a bar. Longer bars usually indicate more meaningful or stable patterns.
